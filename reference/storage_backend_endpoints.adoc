= Astra API (ACS) Speicher-Backend-Endpunkte
:allow-uri-read: 


Die Storage Backend API bietet Zugriff auf Repräsentationen von Anbietern von Speicherdiensten (Storage-Backends), die von verwalteten Clustern und Anwendungen genutzt werden können.
