= Berechtigung abrufen
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/accounts/{account_id}/core/v1/entitlements/{entitlement_id}`#

Gibt die gespeicherte Entitlement-API-Ressource im JSON-Antworttext zurück.



== Parameter

[cols="5*"]
|===
| Name | Typ | In | Erforderlich | Beschreibung 


| Konto-ID | Zeichenkette | Pfad | True  a| 
ID der übergeordneten Kontoressource

* Beispiel: {{.Account}}




| Berechtigungs-ID | Zeichenkette | Pfad | True  a| 
ID der Berechtigungssammlung, die aufgelistet werden soll

|===


== Antwort

[source]
----
Status: 200, Returns the stored Entitlement API resource in the JSON response body.
----
[cols="4*"]
|===
| Name | Typ | Erforderlich | Beschreibung 


| Typ | Zeichenkette | True  a| 
Medientyp der Ressource. Folgende Werte sind definiert:

* "application/astra-entitlement"




| Version | Zeichenkette | True  a| 
Version der Ressource. Folgende Werte sind definiert:

* "1,0"




| id | Zeichenkette | True  a| 
Weltweit eindeutiger Bezeichner der Ressource. Folgende Werte sind definiert:

* Entspricht dem UUIDv4-Schema




| Zuweisung | Zeichenkette | Falsch  a| 
JSON-Zeichenkette mit der ID des Objekts, dem die Berechtigung zugewiesen ist. Nur vorhanden, wenn die Berechtigung zugewiesen ist. Der Zeichenketteninhalt wurde vom Schema nicht validiert.



| Produkt | Zeichenkette | Falsch  a| 
JSON-Zeichenkette mit der aus der Lizenz oder dem Abonnement extrahierten Produktzeichenkette. Der Zeichenketteninhalt wurde vom Schema nicht validiert.



| productVersion | Zeichenkette | Falsch  a| 
JSON-Zeichenkette mit der aus der Lizenz oder dem Abonnement extrahierten Versionszeichenkette. Der Zeichenketteninhalt wurde vom Schema nicht validiert.



| entitlementType | Zeichenkette | True  a| 
JSON-Zeichenkette mit dem Berechtigungstyp. Diese wird aus der Lizenz oder dem Abonnement extrahiert. Der Zeichenketteninhalt wurde vom Schema nicht validiert.



| entitlementValue | Zeichenkette | True  a| 
JSON-Zeichenkette mit der der Berechtigung zugeordneten Kapazität. Der Zeichenketteninhalt wurde vom Schema nicht validiert.



| entitlementConsumption | Zeichenkette | Falsch  a| 
JSON-Zeichenkette mit der aktuell genutzten Kapazität des Berechtigungsanspruchs. Der Zeichenketteninhalt wurde vom Schema nicht validiert.



| sourceLicense | Zeichenkette | Falsch  a| 
JSON-Zeichenfolge mit der Lizenzobjekt-ID, von der die Berechtigung stammt. Folgende Werte sind definiert:

* Entspricht dem Astra-Identifikatorschema




| sourceSubscription | Zeichenkette | Falsch  a| 
JSON-Zeichenfolge mit der ID des Abonnementobjekts, aus dem die Berechtigung stammt. Folgende Werte sind definiert:

* Entspricht dem Astra-Identifikatorschema




| validFromTimestamp | Zeichenkette | Falsch  a| 
JSON-Zeichenkette mit einem Zeitstempel, der angibt, wann die Berechtigung gültig wird. Folgende Werte sind definiert:

* Entspricht dem ISO-8601 Datums- und Zeitschema




| validUntilTimestamp | Zeichenkette | Falsch  a| 
JSON-Zeichenkette mit einem Zeitstempel, der angibt, wann die Berechtigung nicht mehr gültig ist. Folgende Werte sind definiert:

* Entspricht dem ISO-8601 Datums- und Zeitschema




| Metadaten | link:#type_astra_metadata["type_astra_metadata"] | True  a| 
Client- und dienstspezifische Metadaten, die der Ressource zugeordnet sind. Folgende Werte sind definiert:

* Entspricht dem Astra Metadatenschema


|===
