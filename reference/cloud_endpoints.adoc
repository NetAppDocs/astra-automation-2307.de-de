= Astra API (ACS) Cloud-Endpunkte
:allow-uri-read: 


Die Cloud-API ermöglicht den Zugriff auf Cloud-Ressourcen, damit Clients sich mit bestimmten Clouds verbinden können, um Cluster und Anwendungen zu verwalten.
