= Astra API (ACS) Upgrade-Endpunkte
:allow-uri-read: 


Die Upgrade-App-API ermöglicht den Zugriff auf Softwarekomponenten, die für ein Upgrade in Frage kommen, und die Initiierung von Upgrades. Die Verfügbarkeit von Upgrades wird anhand der in der Packages API registrierten Pakete ermittelt.
