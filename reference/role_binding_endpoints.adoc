= Astra API (ACS) Rollenbindungs-Endpunkte
:allow-uri-read: 


Die Role Binding API beschreibt die Beziehungen zwischen den Benutzer-Konto-Paaren bzw. Gruppen-Konto-Paaren sowie die ihnen zugewiesenen Berechtigungen (Rollen).
